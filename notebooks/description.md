# Carpeta `notebooks/`

## Que contiene

Archivos Jupyter Notebook (`.ipynb`) para explorar datos o hacer pruebas.

## Notas

- Esta carpeta es opcional. Sirve para hacer analisis exploratorio antes de
  escribir el codigo final en `src/`.
- Los archivos `.ipynb` se abren con Jupyter Notebook o Jupyter Lab.
- El proyecto principal funciona con los scripts `.py` de `src/`.
